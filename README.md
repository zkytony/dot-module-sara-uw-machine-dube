# dot-module-sara-uw-machine-dube
DUB-E robot setup.

## Installation

1. Make sure `git` is installed on your system

2. Install the [main dot package](https://github.com/pronobis/dot):
    ```
    git clone https://github.com/pronobis/dot.git ~/.dot; ~/.dot/install.sh
    ```

3. Re-login

4. Download this module:
    ```
    dot-get add git@github.com:pronobis/dot-module-sara-uw-machine-dube.git
    ```

5. Install this module:
    ```
    cd ~/.dot/modules/10_dot-module-sara-uw-machine-dube; ./install.sh
    ```
